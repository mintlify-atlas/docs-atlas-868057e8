## Create Vehicle

<api method="POST" endpoint="/vehicles">
  Creates a new vehicle and associates it with a driver profile.

  ### Request Body

  <ParamField body="driverId" type="string" format="uuid" required>
    UUID of the driver who owns the vehicle
  </ParamField>

  <ParamField body="driverProfileId" type="string" format="uuid" required>
    UUID of the driver profile associated with this vehicle
  </ParamField>

  <ParamField body="vehicleTypeId" type="string" format="uuid" required>
    UUID of the vehicle type
  </ParamField>

  <ParamField body="make" type="string" required>
    Vehicle manufacturer (max 50 characters)
  </ParamField>

  <ParamField body="model" type="string" required>
    Vehicle model (max 50 characters)
  </ParamField>

  <ParamField body="year" type="integer" required>
    Manufacturing year
  </ParamField>

  <ParamField body="plateNumber" type="string" required>
    License plate number (max 15 characters)
  </ParamField>

  <ParamField body="color" type="string">
    Vehicle color (max 30 characters)
  </ParamField>

  <ParamField body="capacity" type="integer">
    Passenger capacity (minimum 1)
  </ParamField>

  <ParamField body="isActive" type="boolean">
    Whether the vehicle is active
  </ParamField>

  <ParamField body="status" type="string">
    Vehicle status (enum: VehicleStatus)
  </ParamField>

  <ParamField body="inspectionDate" type="string" format="date">
    Last inspection date (ISO 8601 format)
  </ParamField>

  <ParamField body="lastMaintenanceDate" type="string" format="date">
    Last maintenance date (ISO 8601 format)
  </ParamField>

  <ParamField body="mileage" type="integer">
    Current vehicle mileage
  </ParamField>

  ### Response

  Returns the created vehicle with all its details.

  <ResponseExample>
  ```json
  {
    "success": true,
    "message": "Vehicle created successfully",
    "data": {
      "id": "a1b2c3d4-5678-90ab-cdef-1234567890ab",
      "driverId": "b7a9c9d6-3f28-4e9a-bf82-91f0a8f1a234",
      "driverProfileId": "e6b15a89-3c42-45b8-8d12-84c9c0f22d91",
      "vehicleTypeId": "c4d5e6f7-8901-2345-6789-0abcdef12345",
      "make": "Toyota",
      "model": "Corolla",
      "year": 2022,
      "plateNumber": "ABC-123",
      "color": "Silver",
      "capacity": 4,
      "isActive": true,
      "status": "available",
      "createdAt": "2024-01-15T10:30:00Z",
      "updatedAt": "2024-01-15T10:30:00Z"
    }
  }
  ```
  </ResponseExample>
</api>

<api method="GET" endpoint="/vehicles">
  Retrieves a paginated list of vehicles with optional filters. Admin only.

  ### Query Parameters

  <ParamField query="page" type="integer" default="1">

  </ParamField>

  <ParamField query="limit" type="integer" default="10">

## Delete Vehicle

<api method="DELETE" endpoint="/vehicles/{id}">
  Soft-deletes a vehicle (marks as inactive rather than permanently removing).

  ### Path Parameters

  <ParamField path="id" type="string" format="uuid" required>
    UUID of the vehicle to delete
  </ParamField>

  ### Response

  Returns confirmation of deletion.

  <ResponseExample>
  ```json
  {
    "success": true,
    "message": "Vehicle deleted successfully",
    "data": null
  }
  ```
  </ResponseExample>
</api>
